### TPM

| Op       | Key binding        |
| ------   | ----------------   |
| install  | prefix + I     |
| update   | prefix + U       |
| unintall | prefix + alt + u |

[Github Page](https://github.com/tmux-plugins/tpm)

### Copycat

| Type       | Key binding       |
| ------     | ----------------  |
| file paths | prefix + ctrl - f |
| git path   | prefix + ctrl - g |
| urls       | prefix + ctrl - u |

[Github Page](https://github.com/tmux-plugins/tmux-copycat)
