### Surround

##### Change Surround

```
cs{from}{to}
```

for any tag use t for from

```html
<q>Hello World</q>
cst'
'Hello World'
```

##### Delete Surround

```
ds{surround}
```

##### Add Surround

```
ys{motion}[surround}
```

or on Visual

```
S{surround}
```

for line processing use **ss** instead of **s** for every operation

[Github](https://github.com/tpope/vim-surround)
